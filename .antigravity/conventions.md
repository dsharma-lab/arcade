## Key Conventions

- Files: `snake_case.dart`
- Classes/Widgets: `PascalCase`
- Functions/Variables: `camelCase`
- Global constants: `kCamelCase` (prefixed with `k`)
- One widget class per file (small helpers may coexist)
- Game modules are self-contained: each has `_game.dart`, `_screen.dart`, `_ai.dart`, `_provider.dart`

## Dependency Management

- All dependencies pinned to major.minor in `pubspec.yaml` (e.g. `flame: ^1.18.0`)
- Run `flutter pub upgrade --major-versions` to upgrade, then test
- Never add dependencies without a clear justification in the PR description
