# GameVault 🎮

A free, single-player mobile arcade app built with Flutter and the Flame game engine.
Beat the AI. Beat your score. Have fun.

---

## Games

| Game | Status | Description |
|------|--------|-------------|
| Tic Tac Toe | ✅ R1 | Classic 3×3 grid vs. unbeatable Minimax AI |
| Breakout | 🔜 R2 | Break all the bricks. Don't drop the ball! |
| Snake | 🔜 R2 | Eat, grow, survive. |

---

## Tech Stack

- **Flutter** — cross-platform mobile framework
- **Flame** — 2D game engine for Flutter
- **Riverpod** — state management
- **Hive** — local storage (high scores)
- **go_router** — navigation

## Quick Start

### 1. Install Flutter

```bash
brew install --cask flutter   # macOS
flutter doctor                # fix any issues
```

See `docs/architecture/android-deployment.md` for detailed setup.

### 2. Generate Platform Folders & Install Dependencies

Run this inside the `arcade/` folder to generate the `android/` and `ios/` folders:

```bash
# 1. Generate platform folders (if missing)
flutter create --org com.dsharma_lab --project-name gamevault .

# 2. Fetch dependencies
flutter pub get
```

### 3. Run on Device

```bash
# Connect Android device with USB debugging enabled
flutter devices
flutter run
```

### 4. Run Tests

```bash
flutter test
flutter analyze
```

## Project Structure

```
arcade/
├── .antigravity/      ← AntiGravity AI config (rules, conventions)
├── .agents/           ← Skills, workflows, planning agent prompt
├── docs/
│   ├── prd/           ← Product requirements (overview + per-release)
│   ├── architecture/  ← System design + Android deployment guide
│   ├── api/           ← Future service contracts
│   ├── roadmap/       ← Releases + epics/stories backlog
│   └── sprints/       ← Sprint plans (generated by planning agent)
├── lib/               ← Flutter app source
│   ├── core/          ← theme, router, storage, constants
│   ├── games/         ← one folder per mini-game
│   └── shared/        ← reusable widgets and models
└── test/              ← mirrors lib/ structure
```
